
# SpartaMarket


## 🖥️ 프로젝트 소개


당근마켓이나 중고나라 처럼, 중고품이나, 안 쓰는 제품들을 거래할 수 있는 플리 마켓 사이트


## 🕰️ 개발 기간


24.04.12일(금) - 24.04.19일(금)


## ⚙️ 개발 환경


Framework : Django


Database : Sqlite3


ORM : Django ORM

## 📌 주요 기능


회원가입 / 프로필 이미지 생성


로그인


유저 프로필 페이지


유저 팔로우/팔로잉 기능


로그인 시 쿠키(Cookie) 및 세션(Session) 생성


회원정보 변경 / 프로필 이미지 변경


회원 탈퇴


로그인 시에만 물품 등록 가능


물품 등록 시 내용에 해시 태그 기능


물품 정보 상세 페이지


좋아요 기능


수정, 삭제(CRUD)


최신순, 추천순 정렬


게시글 조회 시 조회수 증가


게시물 제목, 내용, 작성자 검색 가능
